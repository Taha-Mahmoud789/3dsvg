GLB Export & glTF Viewer Verification Workflow
================================================

Prerequisites:
- Dev server running on http://localhost:3000
- glTF Viewer tab open at https://gltf-viewer.donmccurdy.com/
- Save screenshots to test-assets/ folder

Steps:
-------

1. PASTE SVG
   - Select tab 0 (3dsvg editor)
   - Fill the textarea with the SVG code

2. WAIT FOR RENDER
   - Wait 5 seconds for 3D scene to fully render

3. EXPORT GLB
   - Click the download panel button (nth(1) of .pointer-events-auto.absolute.top-5 button)
   - Wait 500ms
   - Click "Download .glb" button
   - Save as test-assets/<name>.glb

4. SWITCH TO glTF VIEWER
   - Select tab 1

5. UPLOAD TO VIEWER
   - Reload https://gltf-viewer.donmccurdy.com/
   - Wait for networkidle
   - Click label[for="file-input"] to trigger file chooser
   - Upload the saved .glb file
   - Wait 3 seconds for model to load

6. SCREENSHOT FRONT
   - Take screenshot -> test-assets/<name>-gltf-front.png

7. ROTATE 180 DEGREES
   - canvas = page.locator('canvas').first()
   - Get bounding box
   - Mouse down at (cx+250, cy)
   - Move 25 steps, each -20px horizontally
   - Mouse up, wait 1 second

8. SCREENSHOT BACK
   - Take screenshot -> test-assets/<name>-gltf-back.png

9. CHECK BOTH SCREENSHOTS
   - Model should be solid (not hollow/invisible) from both angles
   - Compare front vs back view
   - Report any issues found

Example quick-run code:
-----------------------
// Tab 0: Paste SVG
const textarea = page.getByRole('textbox', { name: '<svg xmlns="http://www.w3.org' });
await textarea.fill('PASTE_SVG_HERE');
await page.waitForTimeout(5000);

// Export GLB
await page.locator('.pointer-events-auto.absolute.top-5 button').nth(1).click();
await page.waitForTimeout(500);
const downloadPromise = page.waitForEvent('download', { timeout: 15000 });
await page.getByRole('button', { name: 'Download .glb' }).click();
const download = await downloadPromise;
await download.saveAs('test-assets/output.glb');

// Tab 1: Upload to viewer
await page.locator('[role="tab"]').nth(1).click();
await page.goto('https://gltf-viewer.donmccurdy.com/');
await page.waitForLoadState('networkidle');
const fc = page.waitForEvent('filechooser', { timeout: 5000 });
await page.locator('label[for="file-input"]').click();
(await fc).setFiles('C:\\Users\\tabdo\\Desktop\\3dsvg\\test-assets\\output.glb');
await page.waitForTimeout(3000);

// Front screenshot
await page.screenshot({ path: 'test-assets/output-gltf-front.png', scale: 'device', type: 'png' });

// Rotate 180
const canvas = page.locator('canvas').first();
const box = await canvas.boundingBox();
const cx = box.x + box.width / 2, cy = box.y + box.height / 2;
await page.mouse.move(cx + 250, cy);
await page.mouse.down();
for (let i = 1; i <= 25; i++) await page.mouse.move(cx + 250 - i * 20, cy, { steps: 1 });
await page.mouse.up();
await page.waitForTimeout(1000);

// Back screenshot
await page.screenshot({ path: 'test-assets/output-gltf-back.png', scale: 'device', type: 'png' });
